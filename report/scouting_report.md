\# Centre-Back Scouting Case Study



\## Finding a Replacement Profile for Marc Guéhi



\---



\# Objective



This project identifies statistically similar centre-back profiles using UEFA Euro 2024 performance data.



The objective:



Identify potential replacement profiles for Marc Guéhi based on defensive and possession metrics.



\---



\# Dataset



Competition:

UEFA Euro 2024



Data source:

StatsBomb Open Data



Sample:



\- 51 matches analysed

\- 108 centre-backs identified

\- Minimum threshold: 270 minutes played



\---



\# Methodology



Players were compared using:



\## Defensive metrics



\- Tackles per 90

\- Interceptions per 90

\- Clearances per 90



\## Possession metrics



\- Pass completion percentage

\- Progressive passes per 90



A similarity model was created using normalised statistical distance.



\---



\# Target Player



\## Marc Guéhi



Team:

England



Position:

Centre-back



Profile:



\- Reliable defender

\- Strong positioning

\- Secure passer

\- Comfortable progressing possession



\---



\# Closest Statistical Matches



The model identified the following profiles:



1\. Robin Le Normand — Spain

2\. Jan Vertonghen — Belgium

3\. Riccardo Calafiori — Italy

4\. Wout Faes — Belgium

5\. Jonathan Tah — Germany



\---



\# Player Assessment



\## Robin Le Normand



Strengths:



\- Similar defensive output

\- Strong passing reliability

\- Experienced positioning



Fit:



Closest overall statistical profile.



\---



\## Riccardo Calafiori



Strengths:



\- Progressive passing

\- Ball progression

\- Tactical flexibility



Fit:



Higher-upside modern centre-back profile.



\---



\# Key Findings



The analysis suggests that replacement profiles for Marc Guéhi should combine:



\- Defensive consistency

\- Passing security

\- Progressive ability

\- Tactical adaptability



\---



\# Limitations



The model does not include:



\- Aerial duel quality

\- Defensive context

\- Opposition strength

\- Tracking data

\- Tactical role differences



The output should support scouting decisions rather than replace expert judgement.



\---



\# Conclusion



Data analysis can identify comparable player profiles and support recruitment decisions.



The strongest candidates combine defensive reliability with modern possession ability.
